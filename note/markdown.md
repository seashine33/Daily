# 代码
如果是段落中的代码，可以用`int a = 10;`表示出来。  
如果是整段的代码，可以指定语言  
```C
int a = 10;
for(int i=0; i<10; i++){
    printf("%d", i);
}
```

# 图片
插入图片Markdown语法代码
```

```

注意这是本地图片  
![这是图片](./img/philly-magic-garden.jpg "Magic Gardens")

添加链接  
[![沙漠中的岩石图片](./img/shiprock.jpg "Shiprock")](https://markdown.com.cn)
