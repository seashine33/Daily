# 我为什么要用 VsCode+Markdown+git 来做笔记？
- 我先后用笔记本、Word、幕布和有道云来记笔记，
- 笔记本和Word的缺点类似，容易丢。
- 幕布的缺点就是没有我喜欢的大纲，查找长笔记的时候很费力。
- 有道云的缺点就是不自由，不能很随意的拖动文档。
- VsCode+Markdown+git 的方法就是云的方式有点复杂，
- 但VScode本身就是生产环境，基础的学习过程可以很快得到验证。

# 日志该如何更新？
- 1、task部分
  - 以事件为中心，记录当天的进度，遇见的问题等。
- 2、time
  - 记录随想，最好由标题：内容的形式组成，利于回看。
- 3、note
  - 以问题为标题，答案为内容的格式进行组成。
- 4、game
  - 游戏日志。
